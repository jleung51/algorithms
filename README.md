# sorts
C implementations of various sorts on arrays of integers.

The current list of sorts is as follows:

1. Bogosort
2. Bubble sort
3. Cocktail sort
4. Insertion sort
5. Quick sort
6. Merge sort
7. Selection sort
8. Shell sort
